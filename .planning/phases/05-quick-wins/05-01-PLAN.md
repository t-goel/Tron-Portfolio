In `src/App.jsx`, add a new `useEffect` hook after the existing audio subscription effect (after line 40). This effect registers a global keydown listener for ESC:

```jsx
// NAV-04: ESC exits any active sector back to grid
useEffect(() => {
  function handleKeyDown(e) {
    if (e.key === 'Escape' && useAppState.getState().activeSector !== null) {
      useAppState.getState().setActiveSector(null)
    }
  }
  window.addEventListener('keydown', handleKeyDown)
  return () => window.removeEventListener('keydown', handleKeyDown)
}, [])
```

Key implementation details:
- Empty dependency array `[]` — runs once on mount, never re-registers
- Uses `useAppState.getState()` (NOT the subscribed `activeSector` variable) to avoid stale closures — this matches the existing HUD click handler pattern at lines 128-129
- Guard `!== null` prevents no-op renders when ESC is pressed with no sector open
- Cleanup removes the listener on unmount

Do NOT modify the existing HUD click handler or any other effects. Do NOT bind ESC inside sector components.
