**GatewayPane.jsx** — Self-contained single pane component. Props: `position` ([x, y, z] array), `label` (string like ">_ PROJECTS"), `seed` (number for visual variation).

Structure:
1. **Refs:** `groupRef` (for billboard rotation), `canvasRef` (offscreen 240x160 canvas), `textureRef` (THREE.CanvasTexture), `lastDrawRef` (elapsed time of last texture draw, init to 0).

2. **useEffect (mount):** Create offscreen canvas via `document.createElement('canvas')`, set `width=240`, `height=160`. Create `new THREE.CanvasTexture(canvas)`. Store in refs. Cleanup: dispose texture on unmount.

3. **Idle draw function `drawIdleFrame(ctx, seed)`:**
   - `ctx.fillStyle = '#000000'`; `ctx.fillRect(0, 0, 240, 160)` (clear)
   - Character set: `'0123456789ABCDEF\u2554\u2557\u255A\u255D\u2551\u2550\u2560\u2563\u2566\u2569\u256C01>_/\\'` (hex digits + box-drawing + binary + scattered chars)
   - Font: `ctx.font = "8px 'Roboto Mono', monospace"`
   - Loop 30 columns (x=0..29) x 20 rows (y=0..19):
     - Pick random character from set using `Math.random()` offset by `seed * 0.1`
     - Color distribution: 40% `#00FFFF`, 30% `rgba(255,94,0,0.6)`, 30% `rgba(240,240,240,0.4)` — use `Math.random()` threshold: <0.4 cyan, <0.7 orange, else white
     - Draw at `(col * 8 + 1, row * 8 + 7)` (1px left pad, 7px baseline offset for 8px font in 8px cell)
   - Draw 3-5 random faint wireframe lines: `ctx.strokeStyle = 'rgba(0,255,255,0.15)'`, `ctx.lineWidth = 1`, random start/end points across canvas. Line count = `3 + Math.floor(Math.random() * 3)`.

4. **useFrame (every frame):**
   - **Billboard:** `groupRef.current.rotation.y = Math.atan2(camera.position.x - groupRef.current.position.x, camera.position.z - groupRef.current.position.z)`
   - **Texture throttle:** Check `clock.elapsedTime - lastDrawRef.current > 0.125`. If yes: call `drawIdleFrame(ctx, seed)`, set `textureRef.current.needsUpdate = true`, update `lastDrawRef.current`.

5. **JSX return:**
   ```jsx
   <group ref={groupRef} position={position}>
     <mesh>
       <planeGeometry args={[5, 3.5]} />
       <meshStandardMaterial
         map={textureRef.current}
         color="#00FFFF"
         emissive="#00FFFF"
         emissiveIntensity={0.04}
         transparent
         opacity={0.18}
         side={THREE.DoubleSide}
         toneMapped={false}
       />
     </mesh>
     <lineSegments geometry={edgesGeo}>
       <lineBasicMaterial color="#00FFFF" transparent opacity={0.8} toneMapped={false} />
     </lineSegments>
   </group>
   ```
   Where `edgesGeo` is created via `useMemo(() => new THREE.EdgesGeometry(new THREE.PlaneGeometry(5, 3.5)), [])`.

**GatewayPanes.jsx** — Positions three GatewayPane instances, triggers GSAP rise on phase entry.

1. **Pane config array:**
   ```js
   const PANES = [
     { position: [0, 1.5, -8], label: '>_ PROJECTS', seed: 1 },
     { position: [-6.93, 1.5, 4], label: '>_ ABOUT_ME', seed: 2 },
     { position: [6.93, 1.5, 4], label: '>_ SKILLS', seed: 3 },
   ]
   ```

2. **Pane refs:** `const paneRefs = [useRef(), useRef(), useRef()]` — each passed to GatewayPane as `ref` (use `React.forwardRef` on GatewayPane, forwarding to the `groupRef`).

3. **GSAP rise useEffect:** On mount (since GatewayPanes only renders when phase >= 3 due to Scene.jsx guard):
   ```js
   useEffect(() => {
     paneRefs.forEach((ref) => {
       if (!ref.current) return
       ref.current.position.y = -5  // start below grid
       gsap.to(ref.current.position, {
         y: 1.5,
         duration: 1.5,
         ease: 'power2.out',
       })
     })
   }, [])
   ```
   All three rise simultaneously (no stagger). Start at y=-5 (below grid floor at y=-3), animate to y=1.5.

4. **JSX:** Render three `<GatewayPane>` with position, label, seed, and ref from PANES config.

**Important implementation notes:**
- GatewayPane must use `React.forwardRef` so GatewayPanes can access the group ref for GSAP position animation.
- The `map` property on meshStandardMaterial should be set after textureRef is initialized. Use a state flag or conditional render to avoid passing `null` map on first frame. Alternatively, create the canvas and texture synchronously in a `useMemo` (not useEffect) so it is available on first render.
- The `position` prop on `<group>` sets the initial position, but GSAP animates `ref.current.position.y` directly — this is correct because R3F reads `.position` from the Three.js object, not from the JSX prop after mount.
