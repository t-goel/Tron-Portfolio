- `width: '100%'`, `maxWidth: 680px`, `maxHeight: 480px`, `margin: '0 16px'` (md side padding on mobile)
- `border: '1px solid rgba(0,255,255,0.3)'`, `background: '#000000'`, `borderRadius: 8px`, `overflow: 'hidden'`
- `display: 'flex'`, `flexDirection: 'column'`

**Chrome bar (top):**
- `height: 28px`, `background: 'rgba(20,20,20,0.98)'`, `display: 'flex'`, `alignItems: 'center'`, `paddingLeft: 16px`, `gap: 8px`
- Three dots, each: `width: 12px`, `height: 12px`, `borderRadius: '50%'`, backgrounds: `#FF5F57`, `#FEBC2E`, `#28C840`

**Terminal body:**
- `ref={terminalBodyRef}` for auto-scroll
- `padding: 16px`, `overflowY: 'auto'`, `flex: 1`
- `fontFamily: "'Roboto Mono', monospace"`, `fontSize: 14px`, `color: '#00FFFF'`, `lineHeight: 1.6`

**Typewriter engine:**
- Define `TERMINAL_SEQUENCE` array of objects:
  ```
  [
    { cmd: '$ whoami', output: 'tanmay-goel' },
    { cmd: '$ cat background.txt', output: 'CS at UIUC (Class of 2027). Building full-stack and ML systems since freshman year.' },
    { cmd: '$ cat interests.txt', output: 'AI/ML research, distributed systems, immersive frontend engineering.' },
    { cmd: '$ cat looking_for.txt', output: 'SWE/ML internship opportunities — Summer 2027. Open to research and systems roles.' },
    { cmd: '$ ls contact/', output: 'github.txt  linkedin.txt  email.txt' },
    { cmd: '$ cat contact/*', output: '__CONTACT_LINKS__' },
  ]
  ```
- Use `useRef([])` for `timeoutsRef` to collect all setTimeout IDs
- Use `useState([])` for `lines` — array of `{ type: 'cmd'|'output'|'contact', text: string }` objects
- Use `useState('')` for `currentTyping` — the partially typed command
- Use `useState(false)` for `done` — whether typewriter is complete
- Use `useState(true)` for `cursorVisible` — blinking cursor

**Typing logic (in useEffect on mount):**
- For each sequence entry: schedule setTimeout calls that append characters one at a time to `currentTyping` at 35ms intervals
- After full command is typed: 400ms pause, then push `{ type: 'cmd', text: cmd }` and `{ type: 'output', text: output }` into `lines`, clear `currentTyping`
- For the final `__CONTACT_LINKS__` output: push `{ type: 'contact' }` — rendered specially
- After each state update, `terminalBodyRef.current.scrollTop = terminalBodyRef.current.scrollHeight`
- On complete: `setDone(true)`
- Cleanup: `return () => timeoutsRef.current.forEach(clearTimeout)`

**Cursor blink:**
- CSS keyframes or `setInterval` toggling `cursorVisible` every 500ms — cleanup on unmount
- Cursor character: `▌` in `#00FFFF`

**Contact links rendering:**
- Import `{ contact }` from `../../data/contact.js`
- When a `lines` entry has `type: 'contact'`, render:
  - `<a href={'https://github.com/' + contact.github}>` (or full URL if contact.github starts with http) — text: `github: ${contact.github}`
  - `<a href={'https://linkedin.com/in/' + contact.linkedin}>` — text: `linkedin: ${contact.linkedin}`
  - `<a href={'mailto:' + contact.email}>` — text: `email: ${contact.email}`
  - Each link: `color: '#00FFFF'`, `textShadow: '0 0 8px #00FFFF'`, `textDecoration: 'underline'`, `cursor: 'pointer'`, `target: '_blank'`, `rel: 'noopener noreferrer'`
  - If a value is `'TBD'`, render `[not set]` instead of a link

**Prompt marker:**
- Each command line renders `$` in `#00FFFF` with `letterSpacing: '0.1em'`
- Output lines render in `#F0F0F0` (off-white)
