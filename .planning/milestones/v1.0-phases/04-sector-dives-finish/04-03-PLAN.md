Create `src/components/UI/SkillsSector.jsx` ‚Äî a full-viewport HTML Canvas 2D overlay with interactive network graph.

**Overlay container:**
- Outer div: `position: 'fixed'`, `inset: 0`, `zIndex: 30`, `background: 'rgba(0,0,0,0.90)'`, `backdropFilter: 'blur(12px)'`, `WebkitBackdropFilter: 'blur(12px)'`
- Fade-in: same pattern as ProjectsSector ‚Äî `visible` state + CSS transition `opacity 0.4s`
- `<canvas>` element fills the container: `style={{ width: '100%', height: '100%', display: 'block' }}`

**Canvas setup (useEffect on mount):**
- `canvasRef = useRef(null)`, get `ctx = canvasRef.current.getContext('2d')`
- DPR scaling: `const dpr = window.devicePixelRatio || 1; canvas.width = containerWidth * dpr; canvas.height = containerHeight * dpr; ctx.scale(dpr, dpr)`
- All drawing uses CSS pixel coordinates (not scaled)
- `ResizeObserver` on the container div ‚Äî on resize: recalculate `canvas.width/height`, re-scale ctx, recalculate node positions from new center, redraw

**Data structures (useRef ‚Äî NOT React state):**
- `nodesRef = useRef([])` ‚Äî Tier 1 node objects: `{ id, label, color, x, y, radius: 30, skills: [...], expanded: false }`
- `tier2Ref = useRef([])` ‚Äî Tier 2 nodes: `{ parentId, label, x, y, targetX, targetY, radius: 18, color, visible: false }`
- `racersRef = useRef([])` ‚Äî active racers: `{ fromX, fromY, toX, toY, progress, color, parentId, skillIndex }`
- `hoverRef = useRef(null)` ‚Äî ID of hovered node (Tier 1 or Tier 2)
- `expandedRef = useRef({})` ‚Äî `{ [categoryId]: true/false }`
- `rafRef = useRef(null)` ‚Äî rAF ID

**Tier 1 node layout (computed on mount and resize):**
- 5 nodes in radial arrangement around canvas center
- `centerX = containerWidth / 2`, `centerY = containerHeight / 2`
- `formationRadius = 200` (px)
- For each category at index `i`: `angle = (2 * Math.PI * i / 5) - Math.PI / 2` (start from top), `x = centerX + formationRadius * Math.cos(angle)`, `y = centerY + formationRadius * Math.sin(angle)`

**Hub edges:** Faint lines from canvas center to each Tier 1 node: `ctx.strokeStyle = 'rgba(0,255,255,0.2)'`, `ctx.lineWidth = 1`

**Draw function (`draw(ctx, w, h)`):**
- Clear canvas: `ctx.clearRect(0, 0, w, h)`
- Draw hub edges (center to each Tier 1 node)
- Draw connecting lines from Tier 1 to visible Tier 2 nodes: `ctx.strokeStyle = rgba(nodeColor, 0.4)`, `ctx.lineWidth = 1`
- Draw Tier 1 nodes: filled circle `radius: 30`, `ctx.fillStyle = category.color`, `ctx.arc(x, y, 30, 0, Math.PI*2)`, `ctx.fill()`. If hovered: additional stroke ring `ctx.lineWidth = 3`, `ctx.strokeStyle = category.color`
- Draw Tier 1 labels: `ctx.font = "14px 'Roboto Mono', monospace"`, `ctx.fillStyle = category.color`, `ctx.textAlign = 'center'`, drawn at `y + 30 + 14`
- Draw visible Tier 2 nodes: filled circle `radius: 18`, `ctx.fillStyle = parentColor at 70% opacity`. If hovered: `ctx.lineWidth = 2`, stroke ring
- Draw Tier 2 labels: `ctx.font = "12px 'Roboto Mono', monospace"`, `ctx.fillStyle = parentColor`, `ctx.textAlign = 'left'`, drawn at `x + 22, y + 4`
- Draw active racer trails: for each racer in `racersRef`, draw a line from `(fromX, fromY)` to `(fromX + (toX-fromX)*progress, fromY + (toY-fromY)*progress)`, `ctx.strokeStyle = 'rgba(0,255,255,0.6)'`, `ctx.lineWidth = 2`. Draw a bright dot at the head.

**Click handler (canvas onClick):**
- Convert mouse event to canvas CSS coordinates: `const rect = canvas.getBoundingClientRect(); const mx = e.clientX - rect.left; const my = e.clientY - rect.top;`
- Hit-test each Tier 1 node: `distance = Math.sqrt((mx-node.x)^2 + (my-node.y)^2)`. If `distance <= 30`:
  - If NOT expanded: call `startExpansion(node)` ‚Äî set `expandedRef.current[node.id] = true`, compute Tier 2 target positions (polar offsets from Tier 1 node, distance 80px, evenly spread), create racer objects with `progress: 0`, start rAF loop
  - If expanded: call `startCollapse(node)` ‚Äî create reverse racers from Tier 2 positions back to Tier 1, set `expandedRef.current[node.id] = false`

**Expansion animation (startExpansion):**
- For each skill in the category: compute target position at `80px` from parent node, angle = `(2 * Math.PI * skillIndex / skillCount) + parentAngle`
- Create racer: `{ fromX: node.x, fromY: node.y, toX: targetX, toY: targetY, progress: 0, color: node.color, parentId: node.id, skillIndex }`
- Start rAF loop via `tick(timestamp)`:
  - `elapsed = timestamp - startTime`
  - For each racer: `racer.progress = Math.min(elapsed / 600, 1)` (0.6s = 600ms)
  - If `racer.progress >= 1`: mark corresponding Tier 2 node as visible
  - Call `draw(ctx, w, h)` to render current frame
  - If any racer has `progress < 1`: `rafRef.current = requestAnimationFrame(tick)`
  - Else: `rafRef.current = null`, clear racers array

**Collapse animation (startCollapse):**
- Create reverse racers from Tier 2 positions to Tier 1 node, duration 0.4s (400ms)
- On racer complete: mark Tier 2 node as not visible
- Same rAF loop pattern

**Hover handler (canvas onMouseMove):**
- Hit-test all visible nodes (Tier 1 + visible Tier 2) within 35px
- Update `hoverRef.current` to closest node ID (or null)
- Set `canvas.style.cursor = hoverRef.current ? 'pointer' : 'default'`
- Call `draw()` once to reflect highlight state

**Cleanup (useEffect return):**
- `cancelAnimationFrame(rafRef.current)`
- `resizeObserver.disconnect()`

**Anti-patterns avoided per RESEARCH.md:**
- NO React state for animation progress ‚Äî all animation state in refs
- NO continuous rAF loop ‚Äî only active during expansion/collapse
- Single `rafRef` guard prevents multiple loops piling up
